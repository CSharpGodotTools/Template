## What is this?
An ever expanding utils library for Godot C#. This is the library I am using across all my games, now open source for everyone else to enjoy as well.

## Install
Build the project to generate `GodotUtils.1.0.0.nupkg`. In this repository, the package is copied automatically into `Template/Framework/Libraries`.

Add the local package source to your consuming project and reference the package:

```xml
<PackageReference Include="GodotUtils" Version="1.0.0" />
```
